# 内容

`Hibernate Validator`

包括：

- 基本使用（包`base`）
- 快速失败（包`failfast`）
- 组序列（包`group`）
- 自定义注解（包`custom`）
- AOP（包`aop`）

# 环境

- `spring-boot-starter-validation 2.6.7`
- `spring-boot-starter-aop 2.6.7`
