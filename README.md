# 1 Spring Boot demo

该仓库是`Spring Boot`的各种`Demo`合集，使用`Java`实现。

如果使用`Maven`可以参考旧分支`maven`，但是相关代码并没有更新。

# 2 环境

- `OpenJDK 17.0.3+`
- `Gradle 7.4.1+`
