# 内容

`ShardingSphereJDBC` + `MyBatisPlus` + `MySQL`读写分离和主从复制。

代码包含了读写分离的配置以及测试`Controller`。

运行示例之前需要先准备好`MySQL`主从复制的环境，详情可以参考博客。

# 环境

- `MyBatis Plus 3.5.1`
- `MyBatis Plus Generator 3.5.2`
- `Druid 1.2.10`
- `ShardingSphereJDBC 5.1.1`
- `MySQL 8.0.29`（`Docker`）
